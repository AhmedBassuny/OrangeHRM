Description:
Comprehensive automation framework for OrangeHRM, developed using Selenium WebDriver, Java, and TestNG with the Page Object Model (POM) approach.
This project automates functional scenarios like login, employee management, admin panel validation, and other HR operations with modular, reusable components.

Key Features:

Implements POM for clean separation between test logic and UI elements.

Reusable utilities for driver setup, waits, screenshots, and Excel-based data-driven testing.

TestNG reporting integrated with detailed logs and assertions.

Scalable and easily extendable structure for adding new OrangeHRM modules.

CI/CD-ready with Maven build management and configurable test suites.

Tech Stack: Selenium, Java, TestNG, Maven, POM, Extent Reports
